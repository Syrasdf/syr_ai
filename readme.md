# AI健康助手

这是一个基于人工智能的健康管理助手，能够为用户提供个性化的健康建议、饮食计划和运动指导。

## 主要功能

- 健康数据分析和建议
- 个性化饮食计划推荐
- 科学运动建议
- 心理健康支持
- 睡眠质量分析

## 安装说明

1. 克隆项目到本地
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 配置环境变量
创建.env文件并添加以下内容：
```
OPENAI_API_KEY=你的OpenAI API密钥
```

4. 运行项目
```bash
python main.py
```

## 使用说明

1. 启动服务后，访问 http://localhost:8000/docs 查看API文档
2. 使用API进行健康咨询和获取建议
